First off in making this sprite is to make the image through Photoshop. Three different versions of it was made and then compressed into a single file in a 2103 x 448 png file. Also the canvas size is needed to be focused on the left center. Once made, I install the html file by placing the necessary link and effects on the, style media="screen". First, I labded the id as #bulb and begin to set up a "hover" with the link to move to the second version of the image by subtracting the amount of pixels to the right. The same for the "active" but twice as much. Thus, as the page appears, the first version on the far left of the image shows first while the second version shows upon hovering, and the third and final version shows upon clicking it.
