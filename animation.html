The animation for the css syntax was quite complex overall. Upon making the three images, I got to work in maing the fade in and fade out animation for the images. First, I gave the animation the name of "animationFadeInOut" to be placed in the keyframes. Then, I start to set the certain opacity within each percentage of the cycle. Next, I went into the settings for the animation image, giving the name "animationFadeInOut" to link it with the keyframes, give it a timing function in ease-in-out, the iteration-count is set to infinite, and the animation duration for each cycle will be 6 seconds. Finally, I went into the each images timing on how long it should be delayed. In the form of "img:nth-of-type(#)", I made each gap of the delay to be about 2 seconds apart.
